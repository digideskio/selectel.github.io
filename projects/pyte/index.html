Pyte

VTXXX-совместимый эмулятор терминала, написанный на Python. ХХХ обозначает серию видеотерминалов, разработанных компанией DEC в период 1970-1995. Первым и самым известным является, конечно же, терминал VT100, ныне де-факто являющийся стандартом для всех виртуальных терминалов.

Pyte был создан на основе библиотеки samfoo/vt102, которая в ходе работы была переписана полностью. Мы используем Pyte для показа консолей виртуальныx машин в облаке.

Установка

Клонируйте репозиторий на GitHub: 

$ git clone http://github.com/selectel/pyte

Далее выполните следующую команду: 

$ python setup.py install

Если у вас установлена библиотека setuptools, то установить pyte можно при помощи команды: 

$ easy_install -U pyte

Использование

В Pyte имеется два основных класса: Sсreen и Stream. Screen представляет собой эмулятор экрана терминала. Он хранит в буфере текст и атрибуты текста, отображаемые на экране. Stream представляет собой потоковый процессор, который осуществляет парсинг команд с последующим отображением на экране.

Пример: 

>>> import pyte
>>> screen = pyte.Screen(80, 24)
>>> stream = pyte.Stream()
>>> stream.attach(screen)
>>> stream.feed("Hello World!")
>>> screen.display
   ['Hello World!                                                                    ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ',
    '                                                                                ']

Screen не может читать и изменять переменные окружения, поэтому:

  • выравнивание строк и колонок при событии resize не производится;
  • локальные настройки (например, LC_* и LANG) не могут быть применены;
  • переменная TERM имеет только одно значение - Linux.
