Supload

Утилита на Bash, предназначенная для загрузки файлов в облачные хранилища, использущие OpenStack Swift API. Для работы требует минимум зависимостей и сторонних программ. С ее помощью можно загружать файлы в хранилище, осуществлять рекурсивную загрузку файлов в указанной папке и подпапках, загружать новые файлы, настраивать автоудаление файлов. Может быть также использована для и резервного копирования в хранилище.

Установка

Перед установкой supload на клиентской машине должны быть установлены утилиты curl и file. Они включены в репозитории большинства дистрибутивов Linux и устанавливаются при помощи стандартного менеджера пакетов: 

$ apt-get install curl file

Далее скачайте скрипт, выполнив следующую команду: 

$ wget https://raw.github.com/selectel/supload/master/supload.sh && chmod +x supload.sh

или 

$ curl -Ok https://raw.github.com/selectel/supload/master/supload.sh && chmod +x supload.sh

Можно переместить supload в /usr/bin or /usr/local/bin: 

$ mv supload.sh /usr/local/bin/supload

Примеры использования

Просмотреть мануал: 

$ ./supload.sh

Загрузить один файл (файл будет загружен, если он на текущий момент отсутствует в хранилище или был изменен с момента последней загрузки): 

$ ./supload.sh -u USERNAME -k USERKEY storage_dir local_file.path

Загрузить все файлы из каталога: 

$ ./supload.sh -u USRNAME -k USERKEY -r storage_dir local_dir/

Загрузка без синхронизации (опция -М): 

$ ./supload.sh -u USERNAME -k USERKEY -r -M storage_dir local_dir/

Установить срок, после которого загруженный файл будет удален из хранилища (опция -d): 

$ ./supload.sh -u USERNAME -k USERKEY -d 7d storage_dir file.doc
