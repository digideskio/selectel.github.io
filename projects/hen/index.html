Hen

Hen представляет собой библиотеку на Haskell, реализующую интерфейс к гипервизору Xen. Работает как с третьей, так и с четвёртой версией Хen.

API включает три уровня:

  • System.Xen.Low — реализует интерфейс к вызовам xenctrl;
  • System.Xen.Mid — определяет вспомогательные функции для доступа к System.Xen.Low;
  • System.Xen.High — реализует монаду Xen, обеспечивающую безопасность при выполнении низкоуровневых операций.

Установка

Перед установкой hen необходимо установить заголовки xenctrl. Скачайте и распакуйте любой из указанных ниже пакетов:

Версия XCP Пакет
1.1 xen-devel-3.4.2-1.1.0.704.20055.i686.rpm
1.5 xen-devel-4.1.3-1.6.10.498.23551.i686.rpm

Затем выполните следующую команду: 

$ cabal-dev configure --extra-include-dirs=/path/to/headers

Пример 

module Main (main) where

import System.Xen (runXenT, domainGetInfo)

main :: IO ()
main = print =<< runXenT domainGetInfo
